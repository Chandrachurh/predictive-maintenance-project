# Predictive Maintenance for Engine Failure Detection

This Streamlit application predicts whether an engine is operating normally or requires maintenance based on sensor readings such as:

- Engine RPM
- Lubricating Oil Pressure
- Fuel Pressure
- Coolant Pressure
- Lubricating Oil Temperature
- Coolant Temperature

The application loads the trained machine learning model directly from the Hugging Face Model Hub and provides real-time predictions for proactive maintenance support.
